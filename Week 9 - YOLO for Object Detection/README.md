* Research recent UK data on workers in the construction sector reports indicate 124 workers died in work-related accidents in Great Britain in 2024/25.
* Using the provided dataset on Kaggle, build an object detection model that identifies specified PPE items (hard hat, safety vest, mask etc.) in images.

* The key is that as an AI engineer you think independently and solve the detection problem end-to-end.
